# CS009. Each context manager in a "with" statement must be on its own line
When a `with` statement has multiple context managers, each one must be placed on a separate line using 
parenthesized form or backslashes. 

This approach improves code readability, produces cleaner diffs, and makes it easier to change the code.

### ❌ Anti-pattern
```python
with open('a') as f, open('b'):
    pass
```

```python
with (
    open('a') as f, open('b'),
    open('c'),
):
    pass
```

```python
with (
    open(
        'a',
    ) as f, open('b'),
    open('c'),
):
    pass
```

### ✅ Best practice
```python
with (
    open('a') as f,
    open('b'),
    open('c'),
):
    pass
```
